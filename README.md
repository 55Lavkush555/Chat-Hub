# 💬 ChatHub - Real-Time Chat Application

🚀 **Live Demo:** https://chat-hub--55lavkush555.replit.app/

ChatHub is a modern real-time chat application built with a full-stack JavaScript/TypeScript setup. It allows users to communicate instantly with a clean and responsive UI.

---

## ✨ Features

* 🔐 JWT Authentication (secure login/signup)
* 💬 Real-time messaging using WebSockets
* 🧑 User-based chat system
* ⚡ Fast and responsive UI
* 🌙 Clean and modern interface
* 🔄 Persistent chat storage (MongoDB)

---

## 🛠️ Tech Stack

### 🔹 Frontend

* React / Next.js
* Tailwind CSS

### 🔹 Backend

* Node.js
* Express.js

### 🔹 Database

* MongoDB (Atlas)

### 🔹 Other Tools

* Socket.io (real-time communication)
* JWT (authentication)
* PNPM (package manager)

---


## ⚙️ Environment Variables

Create a `.env` file in root directory:

```
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_super_secret_key
PORT=5000
```
